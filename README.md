# Range Localization & Trajectory Reconstruction

**Кратко.** Этот репозиторий содержит учебно-практическую реализацию двух взаимосвязанных задач:
1. **Локализация по дальномерным измерениям (multilateration)** — решение через линейную МНК (LLS).  
2. **Восстановление траектории по относительным движениям (Pose Graph Optimization, PGO)** — оптимизация графа поз для коррекции дрейфа одометрии.

Репозиторий предназначен для изучения методов оценки состояния и отладки простых экспериментов на синтетических данных.

## Краткое описание модулей

### `localization/multilateration_lls.py`
- Генерирует набор маяков (landmarks) с известными координатами.
- Симулирует измерения расстояний к объекту с аддитивным гауссовским шумом.
- Строит и решает линейную систему вида `A x = B` методом наименьших квадратов (`numpy.linalg.lstsq`) для оценки координат объекта.
- Выводит оценку и сравнивает с истинной позицией.

### `trajectory/pose_graph_optimization.py`
- Строит синтетическую «истинную» траекторию и вычисляет относительные трансформации между последовательными позами.
- Накладывает шум на относительные измерения (одометрия), аккумулирует «шумную» траекторию.
- Формулирует задачу PGO как минимизацию ошибок `r_ij = log(Z_ij^{-1} (T_i^{-1} T_j))`.
- Решает задачу нелинейной МНК с помощью `scipy.optimize.least_squares(method="lm")`.
- Визуализирует: истинную, шумную и оптимизированную траектории.
